# ipl-predictor-2026

A web app built to predict IPL match outcomes, team standings, and player performances for the 2026 season.

---

## Features

* **Match Predictions:** Calculates winners based on form, head-to-head records, and venue history.
* **Interactive UI:** Clean, responsive design built for both desktop and mobile screens.
* **Instant Results:** Fast calculation logic providing real-time prediction scores.
* **2026 Season Stats:** Keeps track of updated team data and metrics.

---

## Tech Stack

* **HTML5** - Semantic markup
* **CSS3** - Custom styling and responsive layouts (Flexbox/Grid)
* **JavaScript (ES6+)** - Prediction logic and DOM manipulation
